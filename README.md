## Frontend

1. cd frontend
2. npm install
3. npm run dev

## Backend

1. cd backend
2. python -m venv store-env
3. store-env\Scripts\activate 
# For Linux/Mac: source store-env/bin/activate
4. pip install Django django-cors-headers
5. python manage.py createsuperuser 
6. cd src
7. python manage.py runserver
